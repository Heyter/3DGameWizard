Description:
------------
  3DGameWizard is a completely free game creator for creating 3D-RPG. 
  The Game Maker is open source and can be used for both not commercial as well as commercial projects. 
  The source code is available under the Zlib license and is written in PureBasic. 
  For the 3D graphics Irrlicht is used.
  The Game Creator provides an easy-to-use editor to create a standalone EXE, which includes all needed files. 
  It is even possible to add your own icon and a logo. 
  The Game Creator currently supports the following objects: walls, gates, various trees, murder holes, bombs,
  flares, keys, treasure chests, lamps, boxes, wolves, knights, fantasy knights, jump fields.
  With the editor it is possible to create huge levels and define game options like you want.


German description:
-------------------
  3DGameWizard ist ein völlig kostenloser Gamecreator zum erstellen von 3D-Rollenspielen.
  Der Gamemaker ist open source und darf sowohl nicht Kommerziell als auch Kommerziell verwendet werden.
  Der Sourcecode steht unter Zlib Lizenz und ist in Purebasic geschrieben.
  Für die 3D-Grafik wird Irrlicht verwendet.
  Der Gamecreator beinhaltet einen einfach zu bedienenden Editor, mit dem eine standalone EXE erstellt werden kann,
  die alle benötigten Dateien beinhaltet.
  Es ist sogar möglich ein Eigenes Icon und ein Eigenes Logo zu integrieren.
  Der Gamecreator unterstützt derzeit folgende Objekte: Wände, Tore, verschiedene Bäume, Mörderlöcher, Tretmienen(Bomben),
  Fackeln, Schlüssel, Schatztruhe, Lampen, boxen, Wölfe, Ritter, Fantasie Ritter, Sprungfelder.
  Mit dem Editor ist es mögliche riesige Levels zu erstellen und die Spieloptionen an seine Wünsche anzupassen.
